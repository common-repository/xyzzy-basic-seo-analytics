XYZZY Basic SEO & Analytics es un sencillo y ligero plugin con el que integrar Analytics y los metadatos SEO en nuestra web.

== Description ==

XYZZY Basic SEO & Analytics es un sencillo y ligero plugin que nos permite integrar Analytics y las funciones básicas de SEO en nuestra web desde la propia interfaz de Wordpress.

¿Qué hace este plugin?

* Agrega un meta box de Herramientas SEO en nuestras entradas, páginas y custom post types
* Permite ajustar palabras clave y meta descripción base desde el panel de administración
* Permite agregar nuestro código de Analytics desde el panel de administración
* Incluye la meta "news_keywords" para la sindicación de Google News
* Utiliza las imágenes destacadas de nuestros posts para la meta "image"
* Incluye las etiquetas meta necesarias para redes sociales

== Requisitos del plugin ==
* Este plugin requiere Wordpress 5.0 o superior
* Este plugin requiere PHP 5.6 o superior

<strong>Compatibilidad con otros plugins de SEO no testada</strong>

== Installation ==

1. Ve a tu escritorio de WP > Plugins y busca por 'XYZZY Basic SEO & Analytics' o…
2. Descarga el plugin del repositorio de Wordpress
3. Sube la carpeta 'xyzzy-basic-seo' al directorio '/wp-content/plugins/'
4. Activa el plugin desde el menú 'Plugins' de Wordpress

== Frequently Asked Questions ==

= ¿Por qué incluimos meta keywords? =

Si bien es cierto que Google ha dejado claro que las keywords no se tienen en cuenta en el posicionamiento, creemos que representan una buena base visual a la hora de sintetizar, ayudando de este modo a generar buenas descripciones SEO. Por otro lado, el atributo 'news_keywords' si que es tenido en cuenta por Google a la hora de incluir las webs sindicadas en Google News, con un límite de diez palabras clave.

Por estos motivos, nuestro plugin permite un máximo de diez keywords. Pero si no quieres que estén basta con que dejes vacío el correspondiente campo de texto :)

= ¿Cómo verifico que funciona? =

1. Entra en tu web
2. Haz click derecho y pulsa la opción 'Inspeccionar' (en Firefox: 'Insepeccionar elemento')
3. Dentro de la ventana de código, despliega la etiqueta 'head'
4. Si todo ha ido bien, deberías encontrar ahí tus metadatos y código de Analytics :)

= ¿Que ganáis con esto? =

Nos gusta contribuir con la comunidad Wordpress, de la misma forma en que otras veces nos nutrimos de ella. Si este plugin te ha gustado solo te pedimos una buena reseña :)
